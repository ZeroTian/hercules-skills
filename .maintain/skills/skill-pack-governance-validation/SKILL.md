# Skill Pack Governance Validation

1. Require visible, tracked, and navigation runtime sets to equal the approved five.
2. Run initializer clone/idempotence/conflict/prerequisite no-mutation tests.
3. Run the nine-environment mocked route/map/fallback/failure matrix.
4. Run all stdlib tests, strict validator, Bash syntax, and diff checks.
5. Stage intended paths and run `.maintain/scripts/check-package.sh`.
6. Confirm secret-like fixtures are rejected without echoing their values.
7. Run `.maintain/scripts/smoke-fresh-clone.sh` on the cumulative staged index.

See `references/usability-and-commit-package-validation.md`. Keep worktree,
staged-index, fresh-clone, and external-facility evidence separate.
